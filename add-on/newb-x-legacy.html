Newb X Legacy

Newb X Legacy, also known as Newb Shader, is a lightweight vanilla-style shader that enhances the default visual quality of Minecraft Bedrock 1.21. It supports Android, Windows, and iOS but requires special patches to function. Please read the installation guide before downloading!

Features

  • Lighting – Soft, natural lighting effects with subtle details.
  • Clouds – Soft clouds that change dynamically during rainy weather.
  • Sky – Bright, natural sky that enhances the shader's beauty. The End also features an improved sky.
  • Water – Calm, relaxing, and visually appealing. During storms, water has realistic wave effects.
  • Underwater – Blue haze with realistic caustics and light streaks.
  • Foliage Wave – Grass, leaves, vines, and crops sway gently in the wind.
  • Lantern Swing – Lanterns sway randomly, a rare feature in Bedrock shaders.
  • Depth – Smooth fog layers make the world feel more expansive.
  • Rain – Realistic rainy weather effects.
  • Entity Glow – Entities emit soft, natural light.
  • Color Correction – Uses tone mapping to create a unique color style.
  • Sun/Moon – A 3D moon with full phases and a smooth yet classic sun.
  • Glowing Ores – Ores shimmer in the dark, making them easier to find.
  • Aurora Borealis – Dynamic aurora lights appear in dark rooms.
  • Glint Effect – Upgraded shimmering effect for enchanted items.

Installation Guide

Disclaimer:

Custom shaders are not officially supported by Minecraft Bedrock. The methods presented here are unofficial workarounds.

Android

  • Install the Patched Minecraft APK.
  • Import the resource pack and activate it in Global Resources.

Windows

  • Activate MaterialBinLoader using BetterRenderDragon.
  • Import the resource pack and activate it in Global Resources.
  • If your Minecraft version is not supported by BetterRenderDragon, use this alternative: Matject Guide.

Linux (x86_64 Architecture)

  • For mcpelauncher-manifest users:
  • Install mcpelauncher-materialbinloader-mod.
  • Import the resource pack and activate it in Global Resources.

iOS

Unfortunately, it is not easy to install shaders on iOS. You would need to jailbreak your device or install an untrusted IPA.
