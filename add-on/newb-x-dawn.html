Newb X Dawn Shader

Newb X Dawn is a beautiful RTX-style shader that enhances your gaming experience, making it more visually stunning and realistic. It supports Android and Windows but requires specific patches to function. Please read the installation instructions before downloading!

Features:

  • Skies similar to those of an RTX package.
  • Beautiful sunset and sunrise.
  • Reflection of the sky on the ground.
  • Water with realistic waves.
  • Shining Armor.
  • Nice End Sky.
  • Lighting slightly similar to RTX.
  • RTX-style volumetric clouds.
  • Beautiful nights and days.
  • Bloom on torches.
  • Glowing particles.
  • End based on the Solas shader from Minecraft Java.
  • Shooting stars and northern lights.
  • Movement in plants, crops, and lanterns.
  • Complementary Clouds Reimagined.
  • Colorful environment based on my favorite RTX package.
  • Variety of shiny blocks.
  • Water depths with sun rays.
  • Lite Version.
  • Sun rays.

Notes:

  • If your mobile device has low or mid-range specs, you should use the Lite Shader Version.
  • If the game runs slowly, try disabling Render Clouds to reduce some lag.
  • The Minecraft apk 1.21.20+ patch is required to support the RenderDragon shader.
  • This shader may not be supported on some devices.

Installation Guide

Disclaimer:

Custom shaders are not officially supported by Minecraft Bedrock. The methods presented here are unofficial workarounds.

Android

  • Install the Patched Minecraft APK.
  • Import the resource pack and activate it in Global Resources.

Windows

  • Activate MaterialBinLoader using BetterRenderDragon.
  • Import the resource pack and activate it in Global Resources.
  • If your Minecraft version is not supported by BetterRenderDragon, use this alternative: Matject Guide.

Linux (x86_64 Architecture)

  • For mcpelauncher-manifest users:
  • Install mcpelauncher-materialbinloader-mod.
  • Import the resource pack and activate it in Global Resources.

iOS

Unfortunately, it is not easy to install shaders on iOS. You would need to jailbreak your device or install an untrusted IPA.
